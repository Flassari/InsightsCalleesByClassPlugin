# Insights Callees by Class Plugin

A plugin for Unreal Insights that adds a "Callees by Class" tab. It's similar to "Callees", except it groups the first level of callees by class, allowing a quick summary of common offenders.

## Installation

Put the plugin in your `Engine/Plugins` directory and rebuild the `UnrealInsights` program. The plugin will automatically load when you launch Unreal Insights. You can open the tab from "Menu -> Timing Insights -> Callees by Class".
